# Post template examples

These draft pages demonstrate each layout. Open the source under `posts/examples/` in the repo while you edit.

- **[Blog post](/posts/examples/example-blog/)** — `layouts/post-blog.njk` (long-form Markdown + HTML: pull quotes, images, captions, full-bleed figures).
- **[Video post](/posts/examples/example-video/)** — `layouts/post-video.njk` (YouTube embed, summary, key ideas, transcript body).
- **[Inspiration post](/posts/examples/example-inspiration/)** — `layouts/post-inspiration.njk` (image + `reflection` text per block, shown as pull quotes; optional Markdown after).

Validation rules for frontmatter live in `scripts/layout-schema.mjs`.
